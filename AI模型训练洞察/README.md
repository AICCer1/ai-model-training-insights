# AI模型训练洞察项目

本项目聚焦主流大模型（GPT、LLaMA、Claude）的训练流程、数据准备与数据格式，按模型隔离文档，便于横向比较与后续扩展。

## 目录结构

- `GPT/` - GPT 系列训练洞察
- `LLaMA/` - LLaMA 系列训练洞察
- `Claude/` - Claude 系列训练洞察
- `common/` - 通用训练方法、数据格式、平台与模型工厂洞察

## 最近补充

### SFT 与 RL 数据资源调研
- `common/sft-rl-data-compute-survey.md`
  - 系统梳理业界主流模型在 SFT 和 RL 阶段使用的数据规模与计算资源
  - 涵盖 InstructGPT、Llama 2/3、DeepSeek-R1、Kimi k1.5、Qwen3 等模型
  - 对比开源与闭源模型的透明度差异，总结长 CoT 推理模型的新范式

### 企业训练平台总览
- `common/enterprise-platform-landscape.md`
  - 用一篇文章集中梳理全球与中国主要企业的大模型训练 / 推理平台形态
  - 重点看：训练平台是否自研、平台是否按钮式、全球与中国分别公开了什么、平台骨架型和服务形态型有什么差异
